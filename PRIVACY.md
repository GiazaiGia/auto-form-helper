# 隐私说明

自动填表助手是 Windows 本地工具。软件没有自建服务器，不会把用户填写的微信号、抖音号、手机号、身份证号、支付宝信息、截图或历史记录上传给作者。

## 本地保存的数据

软件会在用户电脑本地保存以下数据：

- 微信号、抖音号和填表资料
- 本地选择的截图路径
- 填表历史记录
- 监控窗口设置
- 腾讯文档登录所需的本地浏览器资料

安装版默认数据位置在 Windows 用户数据目录下，例如：

```text
%APPDATA%\wechat-order-form-helper\data
```

开发模式下可通过 `FORM_HELPER_DATA_DIR` 指定数据目录。

## 网络访问

软件可能访问以下网络服务：

- 腾讯文档表单页面，用于打开、识别和填写用户提供的表单链接。
- GitHub Releases，用于检查和下载软件更新。

除此之外，软件没有内置作者后台，也不会主动上传用户本地资料。

## 微信相关说明

微信窗口监控依赖本机窗口截图、OCR 或本地微信消息库探测。读取和识别过程在用户电脑本地完成。不同微信版本、Windows 权限、数据库加密状态可能影响识别效果。

## 删除数据

用户可以卸载软件，并手动删除本地数据目录：

```text
%APPDATA%\wechat-order-form-helper
```

删除后，软件内保存的账号资料、历史记录和登录状态会被清空。
